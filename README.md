# Raspberry pi3 model B command Line Setup

## updating the pi
```bash
sudo apt-get update
sudo apt full-upgrade
sudo rpi-update
```

## nodejs
```bash
sudo apt-get install alsa-base alsa-utils
sudo apt install nodejs
```

## AudioSetup
```bash
sudo apt-get install alsa-base alsa-utils
```
